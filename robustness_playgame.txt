@startuml
actor GamePlayer


boundary "Выбор героя" as Select
boundary "Статус лобби" as Sl
boundary "Главный экран" as Main_screen
boundary "Состояние игры" as disp
boundary "Результат" as result

control "Лобби" as lobby
control "Игра" as play


GamePlayer--Main_screen 


Main_screen --> Select: "Кнопка играть"
Select --> lobby
lobby --> Sl
Sl --> play : "Играть"
Sl --> Main_screen : "Выход из лобби"
play --> Main_screen : "Выход из игры"
play --> disp : "Информация о состоянии персонажа"
play --> result : "Результат игры"





note as scenario 
Название прецедента: Playgame
Действующее лицо: game player
Цель: Играть
Предусловия: 
game player прошёл процесс авторизации
 

Главаная последовательность:
1) game player нажимает кнопку играть;
2) система позволяет ему выбрать героя;
3) далее игрок регистрируется в лобби;
4) при заполнении лобби начинается игра;
5) игра продолжается и обновляет интерфейс состояния до последнего живого персонажа игрока;
6) по завершению игры выводится окно с результатами матча.

Альтернативная последовательность 1:
1) game player нажимает кнопку играть;
2) система позволяет ему выбрать героя;
3) далее игрок регистрируется в лобби;
4) game player выходит из лобби.
5) Игрок попадает на главный экран.

Альтернативная последовательность 2:
1) Игрок покидает игру во время матча;
2) Экран подтвержденя выхода;
3) Игрок попадает на главный экран.

Альтернативная последовательность 3:
1) Игрок покидает игру во время матча;
2) Экран подтвержденя выхода;
3) Остаётся в игре.
end note

@enduml
